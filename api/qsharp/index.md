# <a name="q-libraries"></a>Biblioteki języka Q#

Standardowa biblioteka języka Q# obejmuje operacje, funkcje i typy zdefiniowane przez użytkownika służące do opracowywania programów kwantowych. Dołączono także dodatkowe biblioteki, które zostały utworzone na podstawie standardowej biblioteki — biblioteki z zakresu chemii, liczb i uczenia maszynowego.

Aby uzyskać więcej informacji o bibliotekach języka Q#, zobacz <xref:microsoft.quantum.libraries>.
