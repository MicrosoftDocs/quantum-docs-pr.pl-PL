# <a name="enabletestingvianame-user-defined-type"></a>EnableTestingViaName typ zdefiniowany przez użytkownika

Przestrzeń nazw: [Microsoft. Quantum. Diagnostics](xref:Microsoft.Quantum.Diagnostics)

Pakiet: [Microsoft. Quantum. QSharp. Core](https://nuget.org/packages/Microsoft.Quantum.QSharp.Core)


Rozpoznany przez kompilator atrybut, za pomocą którego można zdefiniować alternatywną nazwę, która może być używana podczas ładowania typu lub do celów testowych.

```qsharp

@ Microsoft.Quantum.Core.Attribute()
newtype EnableTestingViaName = (String);
```
