# <a name="equalcp-function"></a>EqualCP, funkcja

Przestrzeń nazw: [Microsoft. Quantum. Logical](xref:Microsoft.Quantum.Logical)

Package [](https://nuget.org/packages/)


Zwraca wartość true, jeśli i tylko wtedy, gdy dwa dane wejściowe są równe.

```qsharp
function EqualCP (a : Microsoft.Quantum.Math.ComplexPolar, b : Microsoft.Quantum.Math.ComplexPolar) : Bool
```


## <a name="input"></a>Dane wejściowe

### <a name="a--complexpolar"></a>Odp.: [ComplexPolar](xref:Microsoft.Quantum.Math.ComplexPolar)

Pierwsza wartość do porównania.


### <a name="b--complexpolar"></a>b: [ComplexPolar](xref:Microsoft.Quantum.Math.ComplexPolar)

Druga wartość do porównania.



## <a name="output--bool"></a>Wynik: [bool](xref:microsoft.quantum.lang-ref.bool)

`true` if i tylko wtedy, gdy `a` jest równa `b` .
