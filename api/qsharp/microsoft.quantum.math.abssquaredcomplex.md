# <a name="abssquaredcomplex-function"></a>AbsSquaredComplex, funkcja

Przestrzeń nazw: [Microsoft. Quantum. Math](xref:Microsoft.Quantum.Math)

Package [](https://nuget.org/packages/)


Zwraca kwadratową wartość bezwzględną złożonej liczby typu `Complex` .

```qsharp
function AbsSquaredComplex (input : Microsoft.Quantum.Math.Complex) : Double
```


## <a name="input"></a>Dane wejściowe

### <a name="input--complex"></a>dane wejściowe: [złożone](xref:Microsoft.Quantum.Math.Complex)

## <a name="output--double"></a>Dane wyjściowe: [Double](xref:microsoft.quantum.lang-ref.double)

Kwadratowa wartość bezwzględna $ | c | ^ 2 = x ^ 2 + y ^ 2 $.
