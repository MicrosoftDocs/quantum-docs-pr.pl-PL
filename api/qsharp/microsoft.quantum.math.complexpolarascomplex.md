# <a name="complexpolarascomplex-function"></a>ComplexPolarAsComplex, funkcja

Przestrzeń nazw: [Microsoft. Quantum. Math](xref:Microsoft.Quantum.Math)

Package [](https://nuget.org/packages/)


Konwertuje liczbę zespoloną typu `ComplexPolar` na liczbę zespoloną typu `Complex` .

```qsharp
function ComplexPolarAsComplex (input : Microsoft.Quantum.Math.ComplexPolar) : Microsoft.Quantum.Math.Complex
```


## <a name="input"></a>Dane wejściowe

### <a name="input--complexpolar"></a>dane wejściowe: [ComplexPolar](xref:Microsoft.Quantum.Math.ComplexPolar)

Liczba złożona $c = r e ^ {i t} $.



## <a name="output--complex"></a>Wynik: [złożony](xref:Microsoft.Quantum.Math.Complex)

Liczba złożona $c = x + i y $.
