# <a name="pluscp-function"></a>PlusCP, funkcja

Przestrzeń nazw: [Microsoft. Quantum. Math](xref:Microsoft.Quantum.Math)

Package [](https://nuget.org/packages/)


Zwraca sumę dwóch danych wejściowych.

```qsharp
function PlusCP (a : Microsoft.Quantum.Math.ComplexPolar, b : Microsoft.Quantum.Math.ComplexPolar) : Microsoft.Quantum.Math.ComplexPolar
```


## <a name="input"></a>Dane wejściowe

### <a name="a--complexpolar"></a>Odp.: [ComplexPolar](xref:Microsoft.Quantum.Math.ComplexPolar)

Pierwsze dane wejściowe $a $ mają być sumowane.


### <a name="b--complexpolar"></a>b: [ComplexPolar](xref:Microsoft.Quantum.Math.ComplexPolar)

Druga wartość wejściowa $b $ do sumowania.



## <a name="output--complexpolar"></a>Wynik: [ComplexPolar](xref:Microsoft.Quantum.Math.ComplexPolar)

Suma $a + b $.
