# <a name="plusi-function"></a>PlusI, funkcja

Przestrzeń nazw: [Microsoft. Quantum. Math](xref:Microsoft.Quantum.Math)

Package [](https://nuget.org/packages/)


Zwraca sumę dwóch danych wejściowych.

```qsharp
function PlusI (a : Int, b : Int) : Int
```


## <a name="input"></a>Dane wejściowe

### <a name="a--int"></a>Odp.: [int](xref:microsoft.quantum.lang-ref.int)

Pierwsze dane wejściowe $a $ mają być sumowane.


### <a name="b--int"></a>b: [int](xref:microsoft.quantum.lang-ref.int)

Druga wartość wejściowa $b $ do sumowania.



## <a name="output--int"></a>Wynik: [int](xref:microsoft.quantum.lang-ref.int)

Suma $a + b $.
