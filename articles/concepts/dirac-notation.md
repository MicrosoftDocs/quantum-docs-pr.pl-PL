# <a name="dirac-notation"></a><span data-ttu-id="24e99-182">Notacja Dirac</span><span class="sxs-lookup"><span data-stu-id="24e99-182">Dirac Notation</span></span>

<span data-ttu-id="24e99-183">Podczas gdy notacja wektora kolumn jest powszechna w algebry liniowym, często jest to bardzo trudne w przypadku przetwarzania w usłudze Quantum, zwłaszcza w przypadku wielu qubits.</span><span class="sxs-lookup"><span data-stu-id="24e99-183">While column vector notation is ubiquitous in linear algebra, it is often cumbersome in quantum computing especially when dealing with multiple qubits.</span></span>  <span data-ttu-id="24e99-184">Na przykład, gdy definiujemy jako $ \psi $ wektor, nie jest wyraźnie jasne, czy $ \psi $ jest wierszem lub wektorem kolumny.</span><span class="sxs-lookup"><span data-stu-id="24e99-184">For example, when we define $\psi$ to be a vector it is not explicitly clear whether $\psi$ is a row or a column vector.</span></span>  <span data-ttu-id="24e99-185">W związku z tym jeśli $ \phi $ i $ \psi $ są wektorami, jest to równie jasne, jeśli $ \phi \psi $ jest jeszcze zdefiniowane, ponieważ kształty $ \phi $ i $ \psi $ mogą być niejasne w kontekście.</span><span class="sxs-lookup"><span data-stu-id="24e99-185">Thus if $\phi$ and $\psi$ are vectors then it is equally unclear if $\phi \psi$ is even defined because the shapes of $\phi$ and $\psi$ may be unclear in the context.</span></span>  <span data-ttu-id="24e99-186">Oprócz niejednoznaczności kształtów wektorów, wyraźne proste wektory korzystające z notacji liniowej algebraicznych wprowadzonej wcześniej mogą być bardzo uciążliwe.</span><span class="sxs-lookup"><span data-stu-id="24e99-186">Beyond the ambiguity about the shapes of vectors, expressing even simple vectors using the linear algebraic notation introduced earlier can be very cumbersome.</span></span> <span data-ttu-id="24e99-187">Na przykład, jeśli chcemy opisać $ $ stan qubit w przypadku, gdy każdy qubit przyjmuje wartość 0, $ $ My wyrażamy stan jako</span><span class="sxs-lookup"><span data-stu-id="24e99-187">For example, if we wish to describe an $n$-qubit state where each qubit takes the value $0$ then we would formally express the state as</span></span> 

<span data-ttu-id="24e99-188">$$\begin{bmatrix}1 \\\\ 0 \end{bmatrix} \otimes \cdots \otimes \begin{bmatrix} \\\\ \end{bmatrix} .</span><span class="sxs-lookup"><span data-stu-id="24e99-188">$$\begin{bmatrix}1 \\\\  0 \end{bmatrix}\otimes \cdots \otimes\begin{bmatrix}1 \\\\  0 \end{bmatrix}.</span></span> $$  

<span data-ttu-id="24e99-189">Ocenianie tego produktu dwuskładnikowego jest niepraktyczne, ponieważ wektor bazuje na wykładniczo dużej przestrzeni i dlatego ten zapis jest w rzeczywistości najlepszym opisem stanu, który można podać przy użyciu poprzedniej notacji.</span><span class="sxs-lookup"><span data-stu-id="24e99-189">Of course evaluating this tensor product is impractical because the vector lies in an exponentially large space and so this notation is in fact the best description of the state that can be given using the previous notation.</span></span>  

<span data-ttu-id="24e99-190">[*Notacja Dirac*](https://en.wikipedia.org/wiki/Bra%E2%80%93ket_notation) rozwiązuje te problemy poprzez przedprezentowanie nowego języka w celu dopasowania do precyzyjnej potrzeby Quantum Mechanics.</span><span class="sxs-lookup"><span data-stu-id="24e99-190">[*Dirac notation*](https://en.wikipedia.org/wiki/Bra%E2%80%93ket_notation) solves these issues by presenting a new language to fit the precise needs of quantum mechanics.</span></span>  <span data-ttu-id="24e99-191">Z tego powodu zaleca się, aby czytelnik nie przeglądał przykładów w tej sekcji jako sztywna recepta opisującą sposób opisywania Stanów Quantum, ale zachęca czytelnika do wyświetlania takich rozwiązań jak sugestie, których można użyć do zwięzłego wyrażania koncepcji Quantum.</span><span class="sxs-lookup"><span data-stu-id="24e99-191">For this reason, we recommend the reader not view the examples in this section as a rigid prescription of how to describe quantum states, but rather encourage the reader to view these as suggestions that can be used to concisely express quantum ideas.</span></span>

<span data-ttu-id="24e99-192">Istnieją dwa typy wektorów w notacji Dirac: wektor *bra* i wektor *KET* , tak jak w przypadku łączenia się z nimi w formie *hamulca* lub wewnętrznego produktu.</span><span class="sxs-lookup"><span data-stu-id="24e99-192">There are two types of vectors in Dirac notation: the *bra* vector and the *ket* vector, so named because when put together they form a *braket* or inner product.</span></span>  <span data-ttu-id="24e99-193">Jeśli $ \psi $ jest wektorem kolumn, możemy napisać go w notacji Dirac jako $ \ket { \psi } $ , gdzie \cdot wskazuje, $ \ket { } $ że jest to wektor kolumny jednostki, tj. wektor *KET* .</span><span class="sxs-lookup"><span data-stu-id="24e99-193">If $\psi$ is a column vector then we can write it in Dirac notation as $\ket{\psi}$, where the $\ket{\cdot}$ denotes that it is a unit column vector, i.e., a *ket* vector.</span></span>  <span data-ttu-id="24e99-194">Podobnie wektor wiersza $ \psi ^ \dagger $ jest wyrażony jako $ \bra { \psi } $ .</span><span class="sxs-lookup"><span data-stu-id="24e99-194">Similarly, the row vector $\psi^\dagger$ is expressed as $\bra{\psi}$.</span></span> <span data-ttu-id="24e99-195">Innymi słowy, $ \psi ^ \dagger $ jest uzyskiwana przez zastosowanie złożonej sprzężonej wartości wejściowej do elementów transponowanych $ \psi $ .</span><span class="sxs-lookup"><span data-stu-id="24e99-195">In other words, $\psi^\dagger$ is obtained by applying entry-wise complex conjugation to the elements of the transpose of $\psi$.</span></span> <span data-ttu-id="24e99-196">Notacja bra-KET bezpośrednio oznacza, że $ \braket { \psi | \psi } $ jest wewnętrznym produktem wektora $ \psi $ , który jest w definicji $ 1 $ .</span><span class="sxs-lookup"><span data-stu-id="24e99-196">The bra-ket notation directly implies that $\braket{\psi|\psi}$ is the inner product of vector $\psi$ with itself, which is by definition $1$.</span></span>  

<span data-ttu-id="24e99-197">Bardziej ogólnie rzecz biorąc, jeśli $ \psi $ $ \phi $ są wektory stanu Quantum i ich wewnętrzny produkt polega $ \braket { \phi | \psi } $ na tym, że prawdopodobieństwo mierzenia stanu $ \ket { \psi } $ $ \ket { \phi } $ jest $ | \braket { \phi | \psi } | ^ 2 $ .</span><span class="sxs-lookup"><span data-stu-id="24e99-197">More generally, if $\psi$ and $\phi$ are quantum state vectors their inner product is $\braket{\phi|\psi}$ which implies that the probability of measuring the state $\ket{\psi}$ to be $\ket{\phi}$ is $|\braket{\phi|\psi}|^2$.</span></span>  

<span data-ttu-id="24e99-198">Poniższa Konwencja służy do opisywania Stanów Quantum, które kodują wartości zero i jeden (qubit podstawowe Stany obliczeniowe):</span><span class="sxs-lookup"><span data-stu-id="24e99-198">The following convention is used to describe the quantum states that encode the values of zero and one (the single-qubit computational basis states):</span></span>

$$
<span data-ttu-id="24e99-199">\begin{bmatrix}1 \\\\ 0 \end{bmatrix} = \ket { } ,\qquad</span><span class="sxs-lookup"><span data-stu-id="24e99-199">\begin{bmatrix} 1 \\\\  0 \end{bmatrix} = \ket{0},\qquad</span></span>
<span data-ttu-id="24e99-200">\begin{bmatrix}0 \\\\ 1 \end{bmatrix} = \ket { } .</span><span class="sxs-lookup"><span data-stu-id="24e99-200">\begin{bmatrix} 0 \\\\  1 \end{bmatrix} = \ket{1}.</span></span>
$$

### <a name="example-representing-the-hadamard-operation-with-dirac-notation"></a><span data-ttu-id="24e99-201">Przykład: reprezentacja operacji Hadamard z notacją Dirac</span><span class="sxs-lookup"><span data-stu-id="24e99-201">Example: representing the Hadamard operation with Dirac notation</span></span>

<span data-ttu-id="24e99-202">Poniższy zapis jest często używany do opisywania Stanów, które wynikają z zastosowania bramy Hadamard do wartości $ \ket { 0 } $ i $ \ket { 1 } $ (które odpowiadają wektorom jednostek w $ $ kierunkach + x i $ -x $ w sferze Bloch):</span><span class="sxs-lookup"><span data-stu-id="24e99-202">The following notation is often used to describe the states that result from applying the Hadamard gate to $\ket{0}$ and $\ket{1}$ (which correspond to the unit vectors in the $+x$ and $-x$ directions on the Bloch sphere):</span></span>

$$
<span data-ttu-id="24e99-203">\frac{1 } { \sqrt { 2 1 } } \begin{bmatrix} \\\\ \end{bmatrix} = H \ket { 0 } = \ket { + } ,\qquad</span><span class="sxs-lookup"><span data-stu-id="24e99-203">\frac{1}{\sqrt{2}}\begin{bmatrix} 1 \\\\  1 \end{bmatrix}=H\ket{0} = \ket{+},\qquad</span></span>
<span data-ttu-id="24e99-204">\frac{1 } { \sqrt { 2 } } \begin{bmatrix} 1 \\\\ -1 \end{bmatrix} = H \ket { 1 } = \ket { - } .</span><span class="sxs-lookup"><span data-stu-id="24e99-204">\frac{1}{\sqrt{2}}\begin{bmatrix} 1 \\\\  -1 \end{bmatrix} =H\ket{1} = \ket{-} .</span></span>
$$

<span data-ttu-id="24e99-205">Te Stany można także rozwijać przy użyciu notacji Dirac jako kwoty $ \ket { 0 } $ i $ \ket { 1 } $ :</span><span class="sxs-lookup"><span data-stu-id="24e99-205">These states can also be expanded using Dirac notation as sums of $\ket{0}$ and $\ket{1}$:</span></span>

$$
<span data-ttu-id="24e99-206">\ket{+}= \frac{ 1 } { \sqrt { 2 } } ( \ket { 0 }  +  \ket { 1 } ), \qquad \ket { - } = \frac { 1 } { \sqrt { 2 } } ( \ket { 0 }  -  \ket { 1 } ).</span><span class="sxs-lookup"><span data-stu-id="24e99-206">\ket{+} = \frac{1}{\sqrt{2}}(\ket{0} + \ket{1}),\qquad \ket{-} = \frac{1}{\sqrt{2}}(\ket{0} - \ket{1}).</span></span>
